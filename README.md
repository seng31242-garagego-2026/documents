# 📄 GarageGo – Documents

This repository contains all formal documentation for the GarageGo project.

## Contents

| Folder | Contents |
|---|---|
| `srs/` | Software Requirements Specification |
| `sds/` | Software Design Specification |
| `diagrams/` | UML, context, use case, activity diagrams |
| `meeting-notes/` | Weekly team meeting notes |
| `approval-forms/` | Project approval and sign-off forms |
| `presentations/` | Slides for presentations and pitches |
| `final-report/` | Final project report |

## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) before making changes.

## Branch Naming

Use `draft/<document-name>` for new documents.
Example: `draft/srs-chapter-2`
